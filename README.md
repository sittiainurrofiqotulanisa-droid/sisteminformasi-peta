# Sistem Informasi Pemetaan Wilayah

## Deskripsi

Sistem Informasi Pemetaan Wilayah merupakan aplikasi berbasis CodeIgniter 4 yang digunakan untuk mengelola data wilayah dan menampilkan informasi pemetaan.

## Fitur

* Login Admin
* Kelola Data Wilayah
* Pemetaan Lokasi
* Dashboard
* Laporan

## Teknologi

* PHP 8.2
* CodeIgniter 4
* MySQL
* Bootstrap
* Leaflet

## Cara Menjalankan

1. Clone repository
2. Jalankan `composer install`
3. Salin file `env` menjadi `.env`
4. Atur konfigurasi database
5. Jalankan `php spark serve`
